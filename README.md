# FlatUI project
Набор элементов UI и демонстрационный сайт.

## Установка и запуск
Для установки и запуска проекта требуется Node.js.  
Из корневой папки скачанного проекта запустите следующую команду:
```
npm install
```
Запуск осуществляется с помощью команды:
```
npm start
```
